# Incident Response

This document outlines the incident response plan for Ninpo Snacks.

## Reporting an Incident

Incidents can be reported by creating an issue in the project's issue tracker.

## Triage

Incidents are triaged by the on-call engineer. The severity of the incident is assessed and a priority is assigned.

## Resolution

The on-call engineer is responsible for resolving the incident. The incident is tracked in the issue tracker and a post-mortem is conducted after the incident is resolved.
