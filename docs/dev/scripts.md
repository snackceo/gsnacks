# Scripts

This document describes the available scripts in the project.

## Frontend Scripts

- `npm run dev`: Starts the frontend development server.
- `npm run build`: Builds the frontend for production.
- `npm run preview`: Previews the production build locally.
- `npm run lint`: Lints the frontend code.
- `npm run typecheck`: Type-checks the frontend code.
- `npm run test`: Runs the frontend tests.
- `npm run glossary:audit`: Audits the glossary.
- `npm run docs:check-links`: Checks for broken links in the documentation.

## Backend Scripts

The backend scripts are located in the `server/scripts` directory. They are run using `node`. For example:

`node server/scripts/add-stores.js`
