# Development Setup

## Prerequisites

- Node.js (v18 or later)
- npm or yarn
- MongoDB
- Git

## Installation

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd gsnacks
    ```

2.  **Install backend dependencies:**
    ```bash
    cd server
    npm install
    ```

3.  **Install frontend dependencies:**
    ```bash
    cd ../src
    npm install
    ```
